This Laravel package gives you the ability to handle simple localization in your application. It provides a set of helpers so you can basically do stuff like:

```
GET /en/about # Displays the about page in english
GET /fr/a-propos # Displays the about page in french
GET /es/a-propos # Displays the about page in french
```

## Installation

You can install the package via composer:

```bash
composer require alexjoffroy/laravel-localization
```
